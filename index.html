Typography

Fira Sans Light (not Fira Sans Regular) is the default font used throughout DLx tools, with a line height of 1.5.

Fira Sans Light Italic is used for italics.

Fira Sans Medium (not Fira Sans Bold) is used for bold styling.

Fira Sans Medium Italic is used for the combination of bold and italics.

Fira Mono is used for monospaced code blocks.

Fira Mono Bold is used for emphasized or bolded code.

Gentium Plus Regular is used for text that needs to support Unicode characters, such as data input fields.

Gentium Plus Italic is used for italicized text that requires Unicode support, such as inline citation of data.

Gentium Fonts are also enlarged slightly so that they can be used inline in stretches of Fira Sans text without too much of a size disparity.

Icons

Each of the following icons is available in the /img folder, and can be linked to via the DLx CDN at https://cdn.digitallinguistics.io/img/{filename}. Some are available in multiple formats. Most icons are black by default, but for some there are versions in different colors as well, following the format {filename}.{color}.{ext}. For icons available in different sizes, the size is indicated by appending the dimensions to the filename, after an underscore, e.g. tunnel_64x64.png. DLx uses Material Design Icons for many of its icons.

  • The box icon is the logo for the DLx app
  • The browse icon is the logo for the DLx data explorer
  • Icon representing programming code, e.g. indicating an area with raw JSON data, etc.

  • The info icon is used for About pages or to indicate where additional information is available
  • A basic menu / navigation icon

  • The GitHub Octicon logo
  • The question icon is used for questions/help pages
  • A generic tools icon
  • The tools icon is used as the logo for the Tools section of the DLx app
  • The JSON tunnel icon is the de facto logo for DLx
  • The blue wug is part of DLx branding, and can be used creatively wherever appropriate
  • The wugbot icon is another de facto logo for DLx, part of DLx branding, and can be used creatively wherever appropriate
