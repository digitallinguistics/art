5 Components

Discrete, self-contained pieces of UI. These styles are not applied automatically, but rather need to be applied using the class with the same name as the element (e.g. apply the .button class to all <button> elements.)

Source: code/code.less, line 3

5.1 Inline Code

The <code> element is used for inline code blocks (use the <pre> element for standalone code blocks).

Example

This is an example of a paragraph with inline code. See the <pre> element for styling for standalone code blocks. 

<p>
  This is an example of a paragraph with <code class=code>inline code</code>. See the <code>&lt;pre&gt;</code> element for styling for standalone code blocks.
</p>

5.2 Inline Elements

Styling for inline elements

Source: inline/abbr/abbr.less, line 1

5.2.1 Abbreviation

Default styling for <abbr> tags, which should be used on the first mention of an abbreviation. It is often accompanied by a <dfn> tag in the same paragraph.

Example

Digital Linguistics (DLx) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag. 

<p>
  <dfn class=dfn>Digital Linguistics</dfn> (<abbr class=abbr>DLx</abbr>) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag.
</p>

Source: inline/cite/cite.less, line 1

5.2.2 Citation

The <cite> element is used for titles of publications, etc.

Example

In his review of Endangered languages and new technologies, Hieber (2015) mentions certain concepts in Digital Linguistics. 

<p>
  In his review of <cite class=cite>Endangered languages and new technologies</cite>, Hieber (2015) mentions certain concepts in Digital Linguistics.
</p>

Source: inline/dfn/dfn.less, line 1

5.2.3 Definition

A <dfn> tag is used the first time an element is defined. The definition of the term should be given in the surrounding text.

Example

Digital Linguistics (DLx) is the science of managing linguistic data, digitally. 

<p>
  <dfn class=dfn>Digital Linguistics</dfn> (<abbr class=abbr>DLx</abbr>) is the science of managing linguistic data, digitally.
</p>

5.2.5 Italic

Used for general italic styling. Other more semantic classes should be preferred over this one when possible. Commonly used with <i> elements (which should also be avoided).

5.4 Pre

Used for standalone code blocks (for inline code blocks, see the <code> tag). If the <pre> element is used by itself, without a <code> element inside it, it will simply display as preformatted text. If a <code> element is included inside the <pre>, however, that element will be display: block;.

Example

This is a <pre> element without a <code> element inside of it.
  This is a <pre> element with a <code> element inside of it.

It is displayed as a standalone block of code.

{

  "dateModified": "2017-03-13T04:51:26.134Z",
  "name": {
    "eng": "Chitimacha",
    "ctm": "Sitimaxa"
  }
}

<pre class=pre>This is a &lt;pre&gt; element without a &lt;code&gt; element inside of it.</pre>

<pre class=pre>
  <code>This is a &lt;pre&gt; element with a &lt;code&gt; element inside of it.

It is displayed as a standalone block of code.

  "name": {
    "eng": "Chitimacha",
    "ctm": "Sitimaxa"
