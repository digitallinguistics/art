1.1 Inline Elements

Styling for inline elements

Source: inline/abbr/abbr.less, line 1

1.1.1 Abbreviation

Default styling for <abbr> tags, which should be used on the first mention of an abbreviation. It is often accompanied by a <dfn> tag in the same paragraph.

Example

Digital Linguistics (DLx) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag. 

<p>
  <dfn class=dfn>Digital Linguistics</dfn> (<abbr class=abbr>DLx</abbr>) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag.
</p>

Source: inline/cite/cite.less, line 1

1.1.2 Citation

The <cite> element is used for titles of publications, etc.

Example

In his review of Endangered languages and new technologies, Hieber (2015) mentions certain concepts in Digital Linguistics. 

<p>
  In his review of <cite class=cite>Endangered languages and new technologies</cite>, Hieber (2015) mentions certain concepts in Digital Linguistics.
</p>

Source: inline/inline.less, line 12

1.1.3 Code Block

The <code> element is used for inline code blocks (use the <pre> element for standalone code blocks).

Example

This is an example of a paragraph with inline code. See the <pre> element for styling for standalone code blocks. 

<p>
  This is an example of a paragraph with <code class=code>inline code</code>. See the <code>&lt;pre&gt;</code> element for styling for standalone code blocks.
</p>

Source: inline/inline.less, line 31

1.1.4 Definition

A <dfn> tag is used the first time an element is defined. The definition of the term should be given in the surrounding text.

Example

Digital Linguistics (DLx) is the science of managing linguistic data, digitally. 

<p>
  <dfn class=dfn>Digital Linguistics</dfn> (<abbr class=abbr>DLx</abbr>) is the science of managing linguistic data, digitally.
</p>

1.1.5 Emphasis

Used for standard emphasis within text.

Example

These words are emphasized. 

<p>
  <em class=em>These words</em> are emphasized.
</p>

Source: inline/inline.less, line 53

1.1.6 Italic

Used for general italic styling. Other more semantic classes should be preferred over this one when possible. Commonly used with <i> elements (which should also be avoided).

Example

This shows an example of italicized text.

<p>This shows an example of <i class=italic>italicized text</i>.</p>

1.1.8 Link

The .link class styles an anchor (<a>) element as an inline link. Change the CSS --link-color variable to change the link's color.

Examples

Default styling

This shows a .link-white link with a dark-colored background.

:hover

When the link is hovered over

This shows a link around a single word, inside a <p> tag.

This shows a .link-white link with a dark-colored background.

:visited

When the link has already been visited

This shows a link around a single word, inside a <p> tag.

This shows a .link-white link with a dark-colored background.

  <a class="link {{modifier_class}}" href="#">This</a> shows a link around a single word, inside a <code>&lt;p&gt;</code> tag.

  <a class="link-white {{modifier_class}}" href="#">This</a> shows a <code>.link-white</code> link with a dark-colored background.
</p>

1.1.9 Strong

Used to draw the reader's attention to a term or concept, indicating that it is important.

Example

This is a paragraph with some strongly emphasized text. 

<p>
  This is a paragraph with some <strong class=strong>strongly emphasized</strong> text.
</p>

1.1.10 Transcriptions

Classes for styling inline transcriptions.

Source: inline/inline.less, line 149

1.1.10.1 Orthographic

This class should be used when citing an orthographic representation of some data, e.g., when writing about writing systems.

Example

This is an example of a sentence with an orthographic representation of some text.

<p>This is an example of a sentence with an <span class=orthographic>orthographic representation</span> of some text.</p>

Source: inline/inline.less, line 172

Used when a Unicode font is needed. More semantic styles such as .phonetic or .phonemic should be preferred over this one when possible.
