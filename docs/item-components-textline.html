6.10 Text Line

A text input styled with a simple underline rather than an inset box. A <p> tag may also be included immediately following the <input> element in order to provide a brief description, explanation, or other help text below the input. NOTE: The :not() selector causes problems for KSS, so that the :disabled variant below isn't rendering correctly; however, it does render correctly in production.

Examples

Default styling

:disabled

For disabled text inputs

:focus

For focused text inputs

:invalid

For invalid data in text inputs

<label class="textline {{modifier_class}}">
  <span>Label Name</span>
  <div>
    <input
      class="{{modifier_class}}"
      inputmode=verbatim
      placeholder='placeholder data'
      type=text
    >
    <p>An abbreviation for this language variety. May contain only letters and numbers.</p>
  </div>
</label>

<label class="textline {{modifier_class}}">
  <span>Label Name</span>
  <div>
    <input
      class="{{modifier_class}}"
      inputmode=verbatim
      pattern=[0-9]
      placeholder='placeholder data'
      type=text
      value='Invalid data'
    >
    <p>An abbreviation for this language variety. May contain only letters and numbers.</p>
  </div>
</label>
