4.1 --bold

The font weight to use for bold styling. Use 600 when bolding text in Fira Sans Regular, and 500 when bolding text in Fira Sans Light. Default: 600.

Border styling. Change the color of the border by changing the --border-color variable within the local scope of the item.

4.4 --border-radius

The value to use for the border-radius rule, set to 0.35em. This should generally not be changed.

4.5 --line-width

Essentially the em equivalent of 1px. Set to 0.075em, which is approximately 1px at a font size of 16px (the default for most browsers). Useful for declaring border width relative to the font size.

4.11 --shadow-button

The box shadows to use on buttons and other button-like elements. The styling is color-agnostic, and works by applying partially transparent box shadows, so it can be applied to elements of any color.

4.14 --underline

Default underlining (using border-bottom). The underline color is unset. You may need to set it explicitly by using border-color-bottom. Usually, however, the underline will wind up being the same color as the font.

4.15 --underline-transparent

The styling for a transparent underline (avoids CSS flicker effects when adding an underline on :hover).
