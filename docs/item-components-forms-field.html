5.2.2 Text Field

Styles an element as a form field.

Examples

Default styling

:disabled

When the field is disabled

:focus

When the field is focused

<input
  class="field {{modifier_class}}"
  type=text
  style="width: calc(50% - 2em);"
  value='This text input has been styled with the .field class'
>

<input
  class="field {{modifier_class}}"
  pattern=[0-9]
  style="width: calc(50% - 2em);"
  type=text
  value='The data in this field is invalid'
>
