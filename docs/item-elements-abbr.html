3.1 Abbreviation

Default styling for <abbr> tags, which should be used on the first mention of an abbreviation. It is often accompanied by a <dfn> tag in the same paragraph.

Example

Digital Linguistics (DLx) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag. 

<p>
  <dfn class=dfn>Digital Linguistics</dfn> (<abbr class=abbr>DLx</abbr>) is the science of managing linguistic data, digitally. This is an example of a paragraph with an abbreviation tag.
</p>
