25.1 Heading 1: Page Title

For page titles. This should be an <h1>, and there should be only one <h1> / .page-title per page. This is also aliased as .page-title.

Example

Page Title (Fira Sans Regular)

Page Title (Fira Sans Light)

<h1 class=page-title>Page Title (Fira Sans Regular)</h1>
<h1 class=page-title style="font-weight: 300;">Page Title (Fira Sans Light)</h1>
