1 Typography

These are the primary fonts used in DLx projects. Other fonts may be used as appropriate when representing linguistic data; however, fonts used for actual DLx interfaces should be limited to the ones shown here.

Source: fonts.less, line 28

1.1 Fira Sans

The Fira Sans set of fonts (developed by Mozilla) are the primary fonts used for DLx interfaces.

Source: fonts.less, line 74

1.1.1 Fira Sans Regular

Example

Fira Sans Regular is the default font used throughout DLx tools, with a line height of 1.5. At larger sizes this font tends to look bold, in which case it may be a good idea to use Fira Sans Light (font-weight: 300;) instead.

<p style="font-family: 'Fira Sans';"><strong>Fira Sans Regular</strong> is the default font used throughout DLx tools, with a line height of 1.5. At larger sizes this font tends to look bold, in which case it may be a good idea to use Fira Sans Light (<code>font-weight: 300;</code>) instead.</p>

1.3 Gentium Plus

The Gentium Plus fonts (developed by SIL) are used for representing linguistic data.

Source: fonts.less, line 244

1.3.1 Gentium Plus Regular

Example

Gentium Plus Regular is used for text that needs to support Unicode characters, such as data input fields. It should typically have font-size: 1.15em; so that it occupies the same line height as Fira Sans.

<p style="font-family: 'Gentium Plus'; font-size: 1.15em;"><strong>Gentium Plus Regular</strong> is used for text that needs to support Unicode characters, such as data input fields. It should typically have <code>font-size: 1.15em;</code> so that it occupies the same line height as Fira Sans.</p>

Source: fonts.less, line 263

1.3.2 Gentium Plus Regular Italic

Example

Gentium Plus Regular Italic is used for italicized text that requires Unicode support, such as inline citation of data. It should typically have font-size: 1.15em; so that it occupies the same line height as Fira Sans.

<p style="font-family: 'Gentium Plus'; font-size: 1.15em; font-style: italic;"><strong>Gentium Plus Regular Italic</strong> is used for italicized text that requires Unicode support, such as inline citation of data. It should typically have <code>font-size: 1.15em;</code> so that it occupies the same line height as Fira Sans.</p>
