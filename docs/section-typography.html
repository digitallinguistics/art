1 Typography

The /fonts folder in this repository contains recommended fonts with good Unicode support. The fonts.less file contains font-family declarations, with fallbacks, for serif and sans-serif fonts. The LinuxLibertine.less file contains @font-face declarations for the Linux Libertine font. You can swap out the font names for your font of choice. All recommend fonts are located in the /fonts folder in .woff2 format (which is supported by all major modern browers).
