5.2 Fieldset

Used for grouping inputs together under a heading (<legend>). Remember that flexbox does not work well with <fieldset>, so you may need to wrap the items inside <fieldset> in a <div>.

Example

Colors

This is an explanatory paragraph for this fieldset.

<fieldset class='fieldset ui form'>

  <legend>Colors</legend>

  <label class='ui radio checkbox'>
    <span>Blue</span>
    <input type=radio name=color>
  </label>

  <label class='ui radio checkbox'>
    <span>Green</span>
    <input type=radio name=color>
  </label>

  <label class='ui radio checkbox'>
    <span>Yellow</span>
    <input type=radio name=color>
  </label>

  <div class='ui radio checkbox'>
    <label>Red</label>
    <input type=radio name=color>
  </div>

  <p class='ui tiny info message'>This is an explanatory paragraph for this fieldset.</p>

</fieldset>
